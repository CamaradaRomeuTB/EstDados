A primeira tela vista pelo usuário, nela o usuário terá uma instrução básica do que deve ser feita no jogo e poderá escolher a dificuldade. 
Fácil: modo 3x3, com duas imagens corretas.
Difícil: modo 5x5, com seis imagens corretas.

2. BOGOSORT
O algoritmo do jogo embaralhará as imagens periodicamente, este período é diferente no modo 3x3 e 5x5. O usuário deverá encontrar todas as imagens corretas sem errar até o algoritmo embaralhar novamente.

3. MODO 3x3
<img width="1920" height="1043" alt="image" src="https://github.com/user-attachments/assets/b6e441da-d144-41f2-83b4-f2b7c5ead350" />
No modo fácil o usuário deverá encontrar os dois elafantes no meio de imagens da savana e de outros animais. O algoritmo de ordenação (BOGOSORT) irá embaralhar todas as imganes a cada um segundo.

4. MODO 5x5
No modo difícil o usuário deverá encontrar seis macacos no meio de imagens da floresta e de outros animais. O algoritmo de ordenação (BOGOSORT) irá embaralhar todas as imagens a cada três segundos.
